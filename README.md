# markus-oberlehner-net
Generator for the personal website of Markus Oberlehner.

## Getting started
- `yarn run init`
- `yarn start`

## Build
- `yarn run build` (development)
- `yarn run build:production` (production)

## Lint
- `yarn run lint`
