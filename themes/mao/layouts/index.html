What I do

It is my passion to tinker around with ideas in my free time. My vision is to create a more awesome way to work with CSS. On the long way to achieve this goal I created several tools to deal with certain aspects of working with CSS.

I built three node-sass custom importer modules and combined them to create the most powerful node-sass custom importer there is: node-sass-magic-importer.

Because I wanted to build CSS in a more modular way, I started to work on avalanche, a package based CSS framework. With the power of node-sass-magic-importer and avalanche it is possible to build robust CSS design systems from well tested packages.
